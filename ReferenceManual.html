Semantic Tags: <a>

Abstract

The anchor tag, or <a>, is a type of element in HTML 5 used to create links. These links can be used to access different websites across the Web, to sites within your Web page, or to positions on sites.

Hyperlinks using <a> can be classified as: absolute links, relative links , and bookmarks.

Details

  • The <a> tag is an anchor element used to create links.
  • These links include:
    • Absolute links-Hyperlinks that contain complete URL references to sites on the Web.
    • Relative links-Links that contain a relative to a URL within your Web site.
    • Bookmarks-Links to content on the same page as the hyperlink.
  • The <a> tag syntax requires the use of the href and id attributes.
    • href is used to contain the URL reference, whether it is absolute or relative.
    • id is used to identify bookmarks (locations of reference) within a document.
  • When using the <a> tag in a absolute or a relative link, always keep in mind that when using links, be sure to distinguish where to put characters when using the < and > expressions.
  • The <a> tag can also be used to import images into a file.

Examples

The first example will show a basic absolute link; that is, one that will redirect you to another website using their URL.

University of Nevada, Reno

In the preceding link, clicking it will take the user to the hompage of the University of Nevada, Reno

The code is as follows: <a href="http://unr.edu/">University of Nevada, Reno</a>

As seen in the code, the <a> tag is put first, and within the tag, the href attribute is used to identify the URL, which in this case is the homepage for UNR. The <a> tag is then closed, completing the link.


The second example will demonstrate a relative link, which will redirect the user to another page within your web site.

Index

This hyperlink will redirect the user to the "Index" page on this site. It is the same as the hyperlink in the menu at the top of the page.

The code is as follows: <a href="Index.html">Index</a>

This code works similar to the absolute link variant, but the URL being referenced must be within your site and named appropriately.


The third example will show how to perform a bookmark, which will lead the user to content on the same page.

Jump to Abstract

This hyperlink will, when clicked, jump to the "Abstract" section of the page. It is the same as the bookmarks below the menu

The code is as follows: <a href="#Abstract">Jump to Abstract</a>

In order for bookmarks to work, an id attribute must be set at the location where the bookmark is needed; in this case, the "Abstract" section. The code is set within the <h2> tag, whichever numbered header it is. In the code, it is: <h2 id="Abstract">Abstract</a>

LIkewise with the other examples, the syntax requires an identifier for reference, but this reference must be connected to a section within the page, as stated before.
